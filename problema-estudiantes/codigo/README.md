# Código del problema de los estudiantes

En esta carpeta se encuentra la implementación en Python del programa de gestión de estudiantes.

El programa permite:
- Registrar nombres y notas de estudiantes
- Calcular el promedio de las notas
- Mostrar los estudiantes con nota mayor o igual al promedio

Este código corresponde al diseño y análisis realizados en el laboratorio de la semana 2.
