# Laboratorio Semana 7 - Pilas y Colas

Repositorio del laboratorio correspondiente a la semana 7 del curso de Estructuras de Datos.

## Objetivo
Implementar y analizar estructuras de datos tipo **pilas y colas**, aplicándolas a problemas prácticos y ejercicios de programación.

## Contenido del repositorio

- **codigo/**  
  Implementación del sistema de atención de pacientes usando colas.

- **notebook/**  
  Notebook con el desarrollo completo del laboratorio, análisis, complejidad y pruebas.

- **hackerrank/**  
  Soluciones a los ejercicios propuestos en la plataforma HackerRank.

- **diagramas/**  
  Diagramas de flujo, secuencia y casos de uso del sistema.

- **pruebas/**  
  Casos de prueba utilizados para validar el funcionamiento del programa.
