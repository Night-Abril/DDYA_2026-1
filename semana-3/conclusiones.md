# Conclusiones

- El uso de Divide y Conquista permitió reducir el tiempo de búsqueda de estudiantes de O(n) a O(log n).

- La aplicación de Programación Dinámica optimizó el cálculo del promedio al almacenar resultados parciales, evitando recomputaciones.

- El análisis de complejidad permitió identificar que la operación más costosa es la eliminación de estudiantes, con un costo O(n).

- El sistema es eficiente para volúmenes moderados de datos y puede escalar utilizando estructuras más avanzadas.
