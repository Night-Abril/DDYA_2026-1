# Diagrama de Caso de Uso – Sistema de Gestión de Estudiantes

## Descripción general
El diagrama de caso de uso representa las funcionalidades principales
del sistema de gestión de estudiantes desde la perspectiva del usuario.

## Actor
Usuario: persona que interactúa con el sistema para administrar la
información de los estudiantes.

## Casos de uso identificados

- Registrar estudiante
- Buscar estudiante
- Actualizar estudiante
- Eliminar estudiante
- Salir del sistema

## Objetivo del diagrama
Mostrar de forma general las acciones que el usuario puede realizar
dentro del sistema sin entrar en detalles de implementación.

Este diagrama permite identificar los requisitos funcionales del sistema.
