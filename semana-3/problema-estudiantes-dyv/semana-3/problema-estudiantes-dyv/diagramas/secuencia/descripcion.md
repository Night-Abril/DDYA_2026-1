# Diagrama de Secuencia – Sistema de Gestión de Estudiantes

## Descripción general
El diagrama de secuencia representa la interacción entre el usuario y el sistema
para realizar las operaciones principales del programa de gestión de estudiantes.

Se muestran los mensajes intercambiados entre:

- Usuario
- Sistema
- Lista de estudiantes

## Flujo representado

1. El usuario selecciona una opción del menú.
2. El sistema solicita los datos necesarios según la operación.
3. El sistema interactúa con la lista de estudiantes para:
   - Registrar un estudiante
   - Buscar un estudiante (búsqueda binaria)
   - Actualizar la información
   - Eliminar un estudiante
4. El sistema retorna el resultado al usuario.

## Objetivo del diagrama
Mostrar el orden en que se ejecutan las operaciones y cómo se comunican
los diferentes componentes del sistema.

Este diagrama permite entender la lógica de interacción del programa
antes de su ejecución.
