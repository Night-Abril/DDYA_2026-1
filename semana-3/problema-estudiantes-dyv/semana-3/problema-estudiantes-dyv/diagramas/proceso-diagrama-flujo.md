# Proceso del diagrama de flujo

El diagrama representa el funcionamiento del sistema de gestión de estudiantes aplicando la estrategia Divide y Vencerás.

Se muestra un menú principal con las operaciones:

- Registrar estudiante
- Buscar estudiante (búsqueda binaria)
- Actualizar estudiante
- Eliminar estudiante
- Salir

Cada operación retorna al menú principal, permitiendo la ejecución continua del sistema hasta que el usuario seleccione la opción de salida.

La búsqueda binaria se utiliza para optimizar el tiempo de localización de estudiantes dentro de una lista ordenada.
