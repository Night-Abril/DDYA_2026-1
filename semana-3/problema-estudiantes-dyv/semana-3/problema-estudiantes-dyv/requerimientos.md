# Requerimientos del sistema (Historias de usuario)

## HU-01
Como usuario del sistema  
quiero registrar un estudiante  
para mantener un control organizado.

## HU-02
Como usuario del sistema  
quiero buscar un estudiante por su nombre  
para acceder rápidamente a su información usando búsqueda binaria.

## HU-03
Como usuario del sistema  
quiero actualizar la información de un estudiante  
para mantener los datos correctos.

## HU-04
Como usuario del sistema  
quiero eliminar un estudiante  
para depurar el sistema.

## HU-05
Como usuario del sistema  
quiero que los estudiantes estén ordenados  
para permitir la aplicación de Divide y Vencerás.
